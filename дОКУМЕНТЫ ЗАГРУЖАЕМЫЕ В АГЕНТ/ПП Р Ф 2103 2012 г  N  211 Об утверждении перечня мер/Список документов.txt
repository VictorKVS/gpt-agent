Список документов, необходимых для выполнения требований Постановления № 211

I. Организационные документы
1. Политика обработки персональных данных
o Определяет цели, порядок обработки и защиты данных.
2. Положение о защите персональных данных
o Устанавливает принципы работы с данными, обязанности сотрудников и меры безопасности.
3. Положение о внутреннем контроле обработки персональных данных
o Описывает порядок проверок соблюдения требований законодательства.
4. Перечень персональных данных, подлежащих обработке
o Включает данные, которые собираются и обрабатываются организацией.
5. Положение об обезличивании персональных данных
o Регламентирует порядок обезличивания данных и использования обезличенной информации.
6. Регламент обработки персональных данных
o Определяет порядок обработки, хранения и уничтожения данных.
7. Положение о работе с физическими носителями данных
o Устанавливает правила работы с бумажными документами, носителями информации (диски, флешки).
8. Инструкция по организации защиты информации
o Руководство для сотрудников по защите данных на всех этапах работы.

II. Документы для сотрудников
9. Должностные инструкции сотрудников
o Описание обязанностей и ответственности сотрудников, работающих с персональными данными.
10. Программа обучения по защите персональных данных
o Курсы и семинары для сотрудников по вопросам защиты данных.
11. Журнал проведения инструктажей
o Учет всех проведённых инструктажей сотрудников.
12. Соглашения о неразглашении персональных данных
o Подписываются всеми сотрудниками, имеющими доступ к данным.

III. Технические и учетные документы
13. Паспорт информационных систем персональных данных (ИСПДн)
o Описание используемых систем, их категории и уровня защищённости.
14. Акт категорирования ИСПДн
o Документ, определяющий категории информационных систем в соответствии с требованиями.
15. Реестр информационных систем, обрабатывающих персональные данные
o Полный список систем, где обрабатываются персональные данные.
16. План мероприятий по защите персональных данных
o Меры, направленные на обеспечение защиты данных.
17. Журнал учёта действий пользователей в ИСПДн
o Записи обо всех действиях сотрудников в системах обработки данных.
18. Журнал инцидентов безопасности
o Фиксация всех нарушений и их последствий.
19. Реестр угроз безопасности персональных данных
o Перечень актуальных угроз и мер по их устранению.
20. План реагирования на инциденты безопасности
o Описание действий при обнаружении инцидента.

IV. Документы для взаимодействия с субъектами данных
21. Образцы уведомлений субъектов данных
o Шаблоны уведомлений о сборе, обработке и уничтожении данных.
22. Регламент рассмотрения запросов субъектов данных
o Порядок обработки запросов на изменение или удаление данных.
23. Согласие на обработку персональных данных
o Форма для получения согласия субъекта на обработку его данных.
24. Регламент отзыва согласия на обработку данных
o Процедура прекращения обработки данных по запросу субъекта.

V. Документы для внешнего взаимодействия
25. Договоры с подрядчиками и контрагентами
o Обязательные пункты о защите персональных данных.
26. Соглашение о защите данных с подрядчиками
o Закрепление ответственности за защиту данных при передаче.
27. Протоколы проверки подрядчиков
o Результаты аудитов подрядчиков на предмет соблюдения требований.

VI. Отчётность и аудиты
28. Уведомление Роскомнадзора об обработке данных
o Документ, поданный в Роскомнадзор с описанием целей и методов обработки.
29. Акт проверки соответствия законодательству
o Результаты внутреннего аудита соответствия требованиям законодательства.
30. Отчёты о выполнении мероприятий по защите данных
o Документ для руководства и надзорных органов.

VII. Документы для обновления и развития
31. План модернизации средств защиты данных
o Стратегия обновления технологий и процессов защиты.
32. План улучшения компетенций сотрудников
o Программа обучения и повышения квалификации.
33. Аналитический отчёт по обработке данных
o Ежегодный отчёт об эффективности мер защиты и выявленных нарушениях.

VIII. Расширенные организационные и контрольные документы
34. Матрица ответственности сотрудников
o Документ, описывающий распределение обязанностей по защите данных между сотрудниками.
35. Положение о внутреннем аудите обработки персональных данных
o Регламентирует порядок проведения внутренних проверок на соответствие требованиям ИБ.
36. Регламент взаимодействия с контролирующими органами
o Описание процедур взаимодействия с Роскомнадзором, ФСБ и ФСТЭК.
37. Акт о назначении ответственных за ИБ
o Документ, фиксирующий назначение сотрудников, ответственных за защиту данных.
38. Инструкция по классификации и уничтожению данных
o Определяет категории данных и процедуры их уничтожения.
39. Регламент работы с данными в условиях удалённого доступа
o Устанавливает требования и ограничения при обработке данных вне офиса.
40. Перечень документов, содержащих персональные данные
o Указание всех документов, хранящих персональные данные (например, анкеты, договора, заявки).

IX. Документы для мониторинга и реагирования
41. План проверки системы обработки данных
o Содержит график и методику проверки эффективности защиты данных.
42. Журнал инцидентов обработки данных
o Учитывает любые случаи, связанные с обработкой, включая ошибки, сбои или нарушения.
43. Протокол расследования инцидента
o Документ, фиксирующий результаты расследования инцидентов и предложенные меры по их устранению.
44. Инструкция по действиям при утечке данных
o Детализированный план действий при выявлении инцидентов с утечкой информации.

X. Документы для работы с подрядчиками и партнёрами
45. Регламент оценки подрядчиков
o Описание методик проверки соответствия подрядчиков требованиям ИБ.
46. Акт передачи данных подрядчику
o Документ, подтверждающий передачу персональных данных подрядчику с указанием объёма и целей обработки.
47. Журнал взаимодействия с подрядчиками
o Фиксация всех операций, связанных с передачей и обработкой данных.

XI. Документы для обучения сотрудников
48. План проведения обучающих мероприятий по ИБ
o График семинаров, тренингов и проверок знаний.
49. Тесты для проверки знаний сотрудников
o Набор вопросов для оценки понимания сотрудниками требований по защите данных.
50. Журнал аттестации сотрудников
o Фиксация результатов проверки знаний сотрудников в области ИБ.

XII. Документы по защите физической инфраструктуры
51. Паспорт помещений, где обрабатываются данные
o Описание всех помещений, где ведётся работа с данными, и применяемых мер защиты.
52. Положение о пропускном режиме в помещениях
o Регламент контроля доступа в офисы и серверные комнаты.
53. Инструкция по действиям при несанкционированном доступе
o План действий при выявлении попыток физического проникновения или утраты носителей данных.
54. Журнал учета доступа к серверным помещениям
o Учет всех сотрудников и посетителей, имевших доступ к серверным помещениям.

XIII. Расширенные технические документы
55. Регламент использования шифрования данных
o Определяет порядок работы с криптографическими средствами защиты.
56. Регламент обновления средств защиты информации
o Описание процедур регулярного обновления программного обеспечения и оборудования.
57. Реестр сертификатов используемых средств защиты
o Список сертифицированных средств защиты, используемых в организации.

XIV. Отчётность и развитие системы защиты
58. Ежегодный отчёт об инцидентах и уязвимостях
o Детализированный анализ всех выявленных инцидентов и уязвимостей за год.
59. План по улучшению системы ИБ
o Стратегия развития технической и организационной защиты данных.
60. Акт об эффективности мер защиты данных
o Документ, фиксирующий результаты анализа защищённости данных и рекомендации по улучшению.
